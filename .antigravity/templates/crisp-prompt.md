# CRISP Prompt Template

Nutze dieses Template für präzise, kontextreiche Prompts:

---

## C - Context
In welchem Modul/Feature befindet sich das Problem?
```
Modul: src/...
UI-Path: window.tab.group.element (F12 ermitteln)
```

## R - Requirements
Was genau soll geändert werden?
```
- Anforderung 1
- Anforderung 2
```

## I - Input
Welche Dateien/UI-Elemente sind betroffen?
```
- Datei 1
- Datei 2
```

## S - Specifications
Welche Constraints gelten?
```
- Keine Breaking Changes
- Kein neues Package installieren
- Performance-kritisch
```

## P - Purpose
Warum ist diese Änderung wichtig?
```
UX-Bug / Performance / Feature
```
