   $ tradingbot-config list-strategies

   Output:

   ┃ Name                       ┃ Type              ┃ Regimes              ┃
   ┡━━━━━━━━━━━━━━━━━━━━━━━━━━━╇━━━━━━━━━━━━━━━━━━╇━━━━━━━━━━━━━━━━━━━━━┩
   │ trend_following_conserv... │ trend_following   │ trend_up, trend_down │
   │ mean_reversion_bb          │ mean_reversion    │ range_bound          │
   └────────────────────────────┴───────────────────┴──────────────────────┘

3. CONVERT STRATEGY TO JSON:
   $ tradingbot-config convert trend_following_conservative \
       --output configs/trend.json

   Output:
   Analyzing strategy: trend_following_conservative
     Entry conditions: 4
     Exit conditions: 3
     Required indicators: 5
     Strategy type: trend_following

   Generating JSON configuration...
   ✓ Configuration saved to: configs/trend.json

4. VALIDATE GENERATED CONFIG:
   $ tradingbot-config validate configs/trend.json

   Output:
   ✓ Configuration valid!
     Indicators: 5
     Regimes: 1
     Strategies: 1

5. USE IN BOT:
   from src.core.tradingbot.bot_controller import BotController
   
   controller = BotController(
       config=bot_config,
       json_config_path="configs/trend.json"
   )

  CLI USAGE EXAMPLES
═══════════════════════════════════════════════════════════════════════════════

# List all strategies in JSON format
tradingbot-config list-strategies --format json

# Convert with options
tradingbot-config convert mean_reversion_bb \
    --output configs/mean_rev.json \
    --no-regime \
    --compact

# Convert multiple strategies
tradingbot-config convert-multiple \
    trend_following_conservative \
    trend_following_aggressive \
    --output configs/trends.json

# Convert all trend strategies (wildcard)
tradingbot-config convert-multiple trend_* \
    --output configs/all_trends.json

# Validate with verbose output
tradingbot-config validate configs/strategy.json --verbose

# List all configs in directory
tradingbot-config list-configs 03_JSON/Trading_Bot/configs --recursive
