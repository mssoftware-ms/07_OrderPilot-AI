Schnellstart

In 5 Minuten loslegen

1. Installation

bash 
# Virtual Environment aktivieren
source .wsl_venv/bin/activate

# Dependencies installieren
pip install -e .

# CLI verfügbar machen
tradingbot-config --help

2. Strategie zu JSON konvertieren

bash 
# Einzelne Strategie konvertieren
tradingbot-config convert trend_following_conservative -o configs/trend.json

# Alle Strategien batch-konvertieren
for strategy in trend_following_conservative trend_following_aggressive \
    mean_reversion_bb mean_reversion_rsi breakout_volatility \
    breakout_momentum momentum_macd scalping_range sideways_range_bounce; do
    tradingbot-config convert "$strategy" -o "configs/$strategy.json"
done

3. JSON Config validieren

bash 
# Config validieren
tradingbot-config validate configs/trend.json

# Ausgabe:
# ✓ Schema validation passed
# ✓ All indicators valid
# ✓ All strategies valid
# ✓ Config ready for use

4. Äquivalenz verifizieren

bash 
# Mit hardcoded vergleichen
tradingbot-config compare configs/trend.json trend_following_conservative

# Ausgabe:
# ✓ Strategies are EQUIVALENT
#   Similarity: 100.0%
#   Entry conditions: ✓ Match
#   Exit conditions: ✓ Match
#   Risk parameters: ✓ Match
